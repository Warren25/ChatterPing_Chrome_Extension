# ChatterPing Privacy Policy

Last updated: 2026-03-15

ChatterPing is a Chrome extension that helps users monitor public online chatter for a configured keyword and view AI-generated summaries.

## Data We Collect

ChatterPing stores the following settings in Chrome sync storage:
- Keyword configured by the user
- Check interval preference
- Notification and summary preferences
- Last check timestamp

ChatterPing sends the configured keyword to the ChatterPing backend API to fetch mention data and generate summaries.

## Data We Do Not Collect

ChatterPing does not collect:
- Account passwords
- Payment information
- Personal contacts
- Browsing history for sale

ChatterPing does not sell personal data.

## How Data Is Used

We use data only to provide extension functionality:
- Fetch mention counts for the configured keyword
- Generate summary text in the popup
- Update extension badge counts

## Data Sharing

Data is processed by:
- ChatterPing backend API
- Third-party AI provider used by the backend for summary generation

Data is not shared for advertising purposes.

## Data Retention

- User settings in Chrome sync storage remain until changed or removed by the user.
- Backend retention follows server-side operational needs and logs.

## User Controls

Users can:
- Change or remove keyword/settings in the extension Settings tab
- Uninstall the extension at any time
- Clear Chrome extension storage through browser settings

## Security

We use HTTPS for production API communication.

## Children's Privacy

ChatterPing is not directed to children under 13.
